# zai-quota

Pi extension that displays your [ZAI Coding Plan](https://z.ai) quota usage as a TUI overlay dashboard.

## Usage

```
/zai-quota
```

Shows an overlay with:

- Current plan level
- Time / token limits with progress bars
- Per-model usage breakdown
- Reset countdown

Press `Escape` or `q` to close.

## Setup

Set the API key in your shell profile:

```bash
export ZAI_API_KEY=your-api-key
```
